    How online multiplayer works
    1. Put your Firebase config in firebase-config.js.
    2. Set ENABLE_MULTIPLAYER = true.
    3. Upload this folder to GitHub Pages.
    4. Click Create online room link.
    5. Send the copied link to your friend.
    6. First browser becomes White. Second browser becomes Black.
