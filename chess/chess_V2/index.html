    No-login multiplayer setup
    1. Create a Firebase project.
    2. Create a Realtime Database.
    3. Do not enable Authentication.
    4. Paste your web app config into firebase-config.js.
    5. Change ENABLE_MULTIPLAYER to true.
    6. Use the public demo rules in firebase-rules.json.
    7. Upload the folder to GitHub Pages.
