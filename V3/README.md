# The Villupuram Run Pro v1.3

This version fixes the Start button issue and is more defensive if a file is missing.

## Very important

Do not open `index.html` alone outside the folder.

Keep this exact structure:

```text
villupuram-run-pro-v13-bulletproof/
├── index.html
├── css/
│   └── styles.css
├── js/
│   ├── config.js
│   └── game.js
└── assets/
    ├── characters/
    │   └── vijay.png
    └── audio/
        └── theme.mp3
```

## Put Vijay image here

```text
assets/characters/vijay.png
```

## Put song here

```text
assets/audio/theme.mp3
```

## Controls

- Jump: Space / Arrow Up / Swipe Up
- Slide: Arrow Down / Swipe Down
- Pause: P / Esc
- Music: M / Music button
